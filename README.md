# patchy — чистильщик ядра для MTK

**patchy** — это Magisk-модуль, который помогает старым телефонам на MediaTek (с ядром >4.14) нормально работать с новыми GSI-прошивками. Проблема обычно в том, что в ядре остались куски кода от фаервола — они мешают интернету. patchy находит эти куски и удаляет их.

## Как это работает (коротко)

- Вы нажимаете **ACTION** в Magisk Manager
- Выбираеnтся `boot.img` образ
- Модуль сам распакует бут, пропатчит ядро и соберёт новый образ
- Готовый файл появится в папке `Download` на телефоне

## Что нужно для работы

- Установленный Magisk (желательно 28+)
- Сам `boot.img` (стоковый или уже пропатченный Magisk) – положите его в корень `/sdcard`
- Немного терпения 🙂

## Установка

1. Скачайте ZIP-архив с [релиза](https://github.com/FunCubeee/patchy/releases)
2. Откройте Magisk Manager → Модули → Установить из хранилища
3. Выберите скачанный ZIP
4. Перезагрузитесь

## Как пользоваться

1. Положите `boot.img` во внутреннюю память (`/sdcard`)
2. Зайдите в Magisk → Модули → нажмите **ACTION** рядом с названием модуля
3. Если файлов `.img` несколько — листайте **громкостью вверх**, выбирайте **громкостью вниз**
4. Ждите сообщения `SUCCESS`
5. Пропатченный образ будет лежать по пути `/sdcard/Download/bpfpatched_дата_время.img`
6. Прошейте его через fastboot или в Magisk (как обычно)

## Откуда взялся бинарник

Внутри модуля есть маленький ARM64-бинарник (около 50 КБ), который делает всю работу. Он скомпилирован из исходников [R0rt1z2/mtk-bpf-patcher](https://github.com/R0rt1z2/mtk-bpf-patcher) через Cython. Python на телефоне **не нужен**.
